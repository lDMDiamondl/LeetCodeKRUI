# LeetCode KR UI 한글화 (LeetCode UI Korean Translator)

## 주요 기능

- **UI 번역**: 버튼, 메뉴, 탭 등 주요 인터페이스 한국어 번역
- **날짜/시간 변환**: 상대 시간(예: 2 days ago → 2일 전)을 한국어 형식으로 변환
- **알고리즘 태그 번역**: 문제 분류 태그(Array → 배열 등) 한국어 번역
- **실시간 적용**: 페이지 전환 및 동적 요소 추가 시 자동 번역 적용
